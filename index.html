MVVM Architecture Refactor

April 2025

Refactored GameViewModel into a cleaner MVVM pattern with extracted UseCases and a dedicated GameSessionController for better separation of concerns.

The architecture now cleanly separates:

  • UndoUseCase - Handles undo/redo logic independently
  • InteractionCoordinator - Manages tile selection and board interaction
  • AutoCompleteUseCase - Powers hint and autocomplete features
  • GameSessionController - Orchestrates the overall game session state

This separation makes the codebase more testable, maintainable, and easier to extend with new features without touching core game logic.

×

3D Board Generator Fix

March 2025

Fixed a critical bug in LayeredBoardGenerator where 3D boards were sometimes generated as fully unsolvable puzzles. This was causing frustration for players who would occasionally hit unwinnable games.

The solution was to rewrite the 3D generation algorithm to use backward generation—the same proven approach that works reliably for 2D boards. Instead of placing tiles forward and hoping they're solvable, we now:

  • Start with a completely clear board
  • Remove tiles backward from a solved state
  • Guarantee every generated board has a solution

Tested across all 3D layouts (PYRAMID, FORTRESS, DRAGON, TURTLE, BRIDGE, CASTLE) with a complete tile count audit to ensure balance.

Camera Controls & Board Interaction

February 2025

Implemented full pinch-to-zoom and pan controls for the 3D game board, giving players complete freedom to position the board however they like.

The tricky part was handling pointer event conflicts—ensuring that pinch gestures don't interfere with tile selection, and that single-tap remains responsive while supporting multi-touch zoom.

The final implementation:

  • Detects zoom intent based on pointer count and movement distance
  • Switches gesture mode cleanly mid-interaction
  • Respects a 3D ray-cast hit test to select tiles even at odd camera angles
  • Provides smooth deceleration on pan release

Players can now explore 3D boards from any angle while maintaining snappy, responsive gameplay.

Welcome Screen & Cherry Blossoms

January 2025

Designed and implemented a beautiful WelcomeScreen with falling cherry blossom petal animations and "Tap to Play" prompt for an elegant first impression.

The animation system uses particle physics to:

  • Generate petals at random positions and times
  • Apply gentle gravity and wind drift
  • Rotate petals naturally as they fall
  • Recycle petals smoothly without visual pop-in

The result is a serene, polished first experience that sets the tone for a thoughtful, crafted game. The animation runs at 60fps without impacting battery life on mobile devices.

Background Location System

December 2024

Built a location metadata system for beautiful background images with on-screen overlays showing where each board photo was taken.

Each background board now carries metadata including:

  • Photographer credit
  • Location name and coordinates
  • Shooting date and camera settings
  • Optional description or story

During gameplay, a subtle overlay displays the location, turning each board into a window to somewhere beautiful in the world. This adds a contemplative, travel-inspired dimension to the puzzle experience.
